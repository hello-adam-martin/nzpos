1. Create `src/lib/csv/parseCSV.ts`:
   ```typescript
   import Papa from 'papaparse'

   export interface ParsedCSV {
     headers: string[]
     rows: Record<string, string>[]
   }

   export function parseCSVText(csvText: string): ParsedCSV {
     const result = Papa.parse<Record<string, string>>(csvText, {
       header: true,
       skipEmptyLines: true,
       transformHeader: (h) => h.trim(),
     })
     return {
       headers: result.meta.fields ?? [],
       rows: result.data,
     }
   }
   ```

   - Export `validateImportRows(rows: Record<string,string>[], mapping: ColumnMapping, existingSKUs: Set<string>, existingCategories: Map<string, string>): ValidatedRow[]`

     a. Extract fields using mapping (e.g., `row[mapping.name]` -> product name)
     b. Convert price via `parsePriceToCents` from `src/lib/money.ts`

     d. If name is empty/missing: mark invalid with "name is required"
     e. If SKU exists in `existingSKUs`: mark as `duplicate`
     f. If category mapped and value present: look up category_id from `existingCategories` map. If not found, mark category name for auto-creation (return it in data, let Server Action handle creation per D-10)
     g. Otherwise: mark as `new` with populated `productData` object (ready for insert)
