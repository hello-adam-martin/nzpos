      ```typescript
      const { data: customers, error } = await adminClient
        .from('customers')
        .select('id, name, email, is_active, created_at, auth_user_id')
        .eq('store_id', storeId)
        .order('created_at', { ascending: false })
      ```
    - Order count: Fetch separately for clean types:
      ```typescript
      const { data: orderCounts } = await adminClient
        .from('orders')
        .select('customer_id')
        .eq('store_id', storeId)
        .in('status', ['completed', 'pending_pickup', 'ready', 'collected', 'refunded'])
      const countMap: Record<string, number> = {}
      for (const o of orderCounts ?? []) {
        if (o.customer_id) countMap[o.customer_id] = (countMap[o.customer_id] ?? 0) + 1
      }
      ```
    - Return: `{ data: customers.map(c => ({ ...c, orderCount: countMap[c.auth_user_id] ?? 0 })) }`
    - Error return: `{ error: "Couldn't load customers. Refresh the page to try again." }`

    **getCustomerDetail.ts** (CUST-03):
    - `'use server'` + `import 'server-only'`
    - Input: `customerId: string` (the `customers.id` UUID)
    - Auth: owner-only, same pattern, get `store_id`
    - Customer query: `adminClient.from('customers').select('id, name, email, is_active, created_at, auth_user_id').eq('id', customerId).eq('store_id', storeId).single()`
    - Orders query: `adminClient.from('orders').select('id, order_number, created_at, total_cents, status').eq('store_id', storeId).eq('customer_id', customer.auth_user_id).order('created_at', { ascending: false })`
    - Return: `{ data: { customer, orders: orders ?? [] } }`

    **disableCustomer.ts** (CUST-04 — CRITICAL two-step operation):
    - `'use server'` + `import 'server-only'`
    - Zod schema: `z.object({ customerId: z.string().uuid() })`
    - Auth: owner-only via `createSupabaseServerClient()`, get `store_id`
    - Step A — DB flag: `createSupabaseAdminClient().from('customers').update({ is_active: false }).eq('id', customerId).eq('store_id', storeId).select('auth_user_id').single()`
    - Check error/null: return `{ error: 'Customer not found' }`
    - Step B — Auth ban: `admin.auth.admin.updateUserById(data.auth_user_id, { ban_duration: '876600h' })`
    - On ban error, ROLLBACK DB: `admin.from('customers').update({ is_active: true }).eq('id', customerId)`
    - Return `{ error: 'Failed to disable account' }` on rollback
    - On success: `revalidatePath('/admin/customers')`, `revalidatePath('/admin/customers/' + customerId)`, return `{ success: true }`

    **enableCustomer.ts** (CUST-04 re-enable):
    - Same structure as disableCustomer but reversed:
    - Step A: `.update({ is_active: true })`
    - Step B: `admin.auth.admin.updateUserById(data.auth_user_id, { ban_duration: 'none' })`
    - Rollback on error: `.update({ is_active: false })`
    - Revalidate same paths, return `{ success: true }`
