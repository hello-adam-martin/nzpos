Create `loyalty_settings` table (Option B from research):
```sql
CREATE TABLE public.loyalty_settings (
  store_id UUID PRIMARY KEY REFERENCES public.stores(id),
  earn_rate_cents INTEGER NULL,
  redeem_rate_cents INTEGER NULL,
  is_active BOOLEAN NOT NULL DEFAULT true,
  created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
  updated_at TIMESTAMPTZ NOT NULL DEFAULT now()
);
```
- `earn_rate_cents`: cents per point earned (e.g., 100 = earn 1pt per $1). NULL = not configured.
- `redeem_rate_cents`: cents value per point for redemption (e.g., 1 = 1pt = $0.01, so 100pts = $1). NULL = not configured.
- `is_active`: merchant pause toggle (D-15). Default true so once rates are set, earning starts.

Create `loyalty_points` table (denormalized balance):
```sql
CREATE TABLE public.loyalty_points (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  store_id UUID NOT NULL REFERENCES public.stores(id),
  customer_id UUID NOT NULL REFERENCES public.customers(id),
  points_balance INTEGER NOT NULL DEFAULT 0 CHECK (points_balance >= 0),
  loyalty_banner_dismissed_at TIMESTAMPTZ NULL,
  created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
  updated_at TIMESTAMPTZ NOT NULL DEFAULT now(),
  UNIQUE (store_id, customer_id)
);
```

Create `loyalty_transactions` table (append-only log):
```sql
CREATE TABLE public.loyalty_transactions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  store_id UUID NOT NULL REFERENCES public.stores(id),
  customer_id UUID NOT NULL REFERENCES public.customers(id),
  points_delta INTEGER NOT NULL,
  balance_after INTEGER NOT NULL CHECK (balance_after >= 0),
  transaction_type TEXT NOT NULL CHECK (transaction_type IN ('earn', 'redeem', 'adjustment')),
  order_id UUID REFERENCES public.orders(id),

  staff_id UUID REFERENCES public.staff(id),
  created_at TIMESTAMPTZ NOT NULL DEFAULT now()
);
```

Create `earn_loyalty_points` RPC (SECURITY DEFINER, service_role only):
- Parameters: `p_store_id UUID, p_customer_id UUID, p_order_id UUID DEFAULT NULL, p_net_amount_cents INTEGER DEFAULT 0, p_channel TEXT DEFAULT 'pos', p_staff_id UUID DEFAULT NULL`
- Returns JSONB: `{ points_earned, balance_after }` or `{ points_earned: 0, reason: 'not_configured' }`
- Logic: look up loyalty_settings, check is_active AND both rates non-null (D-10 gate), calculate `FLOOR(p_net_amount_cents / earn_rate_cents)`, UPSERT loyalty_points, INSERT loyalty_transactions with type 'earn'
- REVOKE from PUBLIC, GRANT to service_role

Create `redeem_loyalty_points` RPC (SECURITY DEFINER, service_role only):
- Parameters: `p_store_id UUID, p_customer_id UUID, p_points_to_redeem INTEGER, p_order_id UUID DEFAULT NULL, p_channel TEXT DEFAULT 'pos', p_staff_id UUID DEFAULT NULL`
- Returns JSONB: `{ discount_cents, balance_after }` or error
- Logic: SELECT FOR UPDATE on loyalty_points row (concurrency protection), validate balance >= points, look up redeem_rate_cents, deduct balance, INSERT loyalty_transactions with type 'redeem'
- REVOKE from PUBLIC, GRANT to service_role

**2. Create `src/lib/loyalty-utils.ts`:**

```typescript
import { formatNZD } from '@/lib/gst'  // or wherever formatNZD lives

export function calculatePointsEarned(netAmountCents: number, earnRateCents: number): number {
  if (netAmountCents <= 0 || earnRateCents <= 0) return 0
  return Math.floor(netAmountCents / earnRateCents)
}

export function calculateRedemptionDiscount(pointsToRedeem: number, redeemRateCents: number): number {
  if (pointsToRedeem <= 0 || redeemRateCents <= 0) return 0
  return pointsToRedeem * redeemRateCents
}

export function formatLoyaltyDisplay(pointsBalance: number, redeemRateCents: number): string {
  const dollarValue = calculateRedemptionDiscount(pointsBalance, redeemRateCents)
  return `${pointsBalance} pts (${formatNZD(dollarValue)} available)`
}
```
