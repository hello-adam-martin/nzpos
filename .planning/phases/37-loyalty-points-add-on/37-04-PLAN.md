**3. Earn loyalty points on net amount:**

Look up customer by email (from `session.customer_details?.email`):

```typescript
// Loyalty points earning — look up customer by email
const customerEmail = session.customer_details?.email
if (customerEmail) {
  const { data: customer } = await supabase
    .from('customers')
    .select('id')
    .eq('store_id', storeId)
    .eq('email', customerEmail)
    .maybeSingle()

  if (customer) {
    // D-09: net amount = total - gift card - loyalty discount
    const totalCents = session.amount_total ?? 0
    const giftCardAmountCents = session.metadata?.gift_card_amount_cents
      ? parseInt(session.metadata.gift_card_amount_cents, 10)
      : 0
    const netAmountCents = totalCents + giftCardAmountCents - loyaltyDiscountCents
    // Note: totalCents from Stripe is already reduced by negative line items
    // gift_card_amount_cents was subtracted as negative line item, so add back then subtract properly
    // Actually: Stripe amount_total = sum of all line items (incl negative ones)
    // So amount_total already has gift card + loyalty deductions baked in
    // For earning: use amount_total directly as net (both deductions already applied)

    if (totalCents > 0) {
      try {
        await supabase.rpc('earn_loyalty_points', {
          p_store_id: storeId,
          p_customer_id: customer.id,
          p_order_id: orderId,
          p_net_amount_cents: totalCents,  // Stripe amount_total already reflects deductions
          p_channel: 'online',
          p_staff_id: null,
        })
      } catch (err) {
        console.warn('[stripe-webhook] Loyalty earning failed (non-fatal):', err)
      }
    }
  }
}
```

**IMPORTANT NOTE on Stripe amount_total:** `session.amount_total` is in the smallest currency unit (cents for NZD) and already reflects all line items including negative ones (gift card deduction, loyalty deduction). So it IS the net amount. Do NOT subtract gift card or loyalty amounts again — they are already factored into `amount_total`.

If this assumption needs verification, read the gift card webhook path to see how `amount_total` is used there.
