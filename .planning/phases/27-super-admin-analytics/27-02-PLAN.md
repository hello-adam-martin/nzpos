    3. **Active subscriptions:** Query `platform_analytics_snapshots` where status IN ('active', 'past_due', 'trialing') and snapshot_month = current month. Use `{ count: 'exact', head: true }`. Display as integer.

Group by addon_type (in JS — Supabase JS client doesn't support GROUP BY). Build `{ name: displayName, mrr: sumCents / 100 }[]`. Display names: `{ xero: 'Xero', inventory: 'Inventory', email_notifications: 'Email Notifications', custom_domain: 'Custom Domain' }`.

    6. **Last synced timestamp:** Query `analytics_sync_metadata` where id = 1, get `last_synced_at`. Pass to AnalyticsContent.
