**A. src/actions/orders/createCheckoutSession.ts — per D-12 and Research Pattern 5:**

1. Add optional fields to the input Zod schema:
   ```typescript
   giftCardCode: z.string().transform(s => s.replace(/\D/g, '')).optional(),
   giftCardAmountCents: z.number().int().positive().optional(),
   ```

2. When `giftCardCode` is provided, validate the gift card:
   - Query gift_cards table: check store_id + code, status = 'active', not expired, balance >= giftCardAmountCents
   - If validation fails, return `{ error: 'gift_card_invalid' }` with specific reason

3. **Full cover path (D-12):** When `giftCardAmountCents >= totalCents`:
   - Do NOT create a Stripe session
   - Create the order directly (insert into orders table with status = 'completed', payment_method = 'gift_card')
   - Call `supabase.rpc('redeem_gift_card', { p_store_id: storeId, p_code: giftCardCode, p_amount_cents: totalCents, p_channel: 'online', p_order_id: orderId })`
   - Decrement stock (call the stock decrement logic or RPC)
   - Send order confirmation email
   - Return `{ redirect: '/store/${slug}/order/${orderId}/confirmation?token=${lookupToken}' }`

4. **Partial cover path:** When `giftCardAmountCents < totalCents`:
   - Compute Stripe amount: `totalCents - giftCardAmountCents`
   - Create Stripe Checkout Session for the reduced amount
   - Store gift card info in Stripe session metadata:
     ```typescript
     metadata: {
       ...existingMetadata,
       gift_card_code: giftCardCode,
       gift_card_amount_cents: String(giftCardAmountCents),
     }
     ```
   - Return `{ url: session.url }` as before
   - CRITICAL: Do NOT deduct gift card balance here (Pitfall 1 — wait for webhook)

**B. src/app/api/webhooks/stripe/route.ts — partial gift card redemption:**

In the existing `checkout.session.completed` handler, after detecting it's NOT a gift_card issuance (metadata.type !== 'gift_card'):

1. Check if `session.metadata?.gift_card_code` exists (partial gift card payment)
2. If yes, after the normal order completion logic:
   - Extract `gift_card_code` and `gift_card_amount_cents` from metadata
   - Call `supabase.rpc('redeem_gift_card', { p_store_id: storeId, p_code: gift_card_code, p_amount_cents: Number(gift_card_amount_cents), p_channel: 'online', p_order_id: orderId })`
   - Log success/failure but don't fail the webhook (gift card redemption failure should not block order completion — log warning)
3. The order's payment_method should reflect the split (e.g., store as 'online' as before — the gift card portion is tracked via gift_card_redemptions table)
