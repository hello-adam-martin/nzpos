Per D-01, D-02, D-03, D-04, ADMIN-01, ADMIN-06:

1. **Auth and store context** (follow promos/page.tsx pattern):
   ```typescript
   import { createSupabaseServerClient } from '@/lib/supabase/server'
   import { redirect } from 'next/navigation'
   const supabase = await createSupabaseServerClient()
   const { data: { user } } = await supabase.auth.getUser()
   const storeId = user?.app_metadata?.store_id as string | undefined
   if (!storeId) redirect('/admin/login')
   ```

2. **Today's orders query:**
   ```typescript
   const todayStart = new Date()
   todayStart.setHours(0, 0, 0, 0)

   const { data: todayOrders } = await supabase
     .from('orders')
     .select('total_cents, channel')
     .eq('store_id', storeId)
     .in('status', ['completed', 'pending_pickup', 'ready', 'collected'])
     .gte('created_at', todayStart.toISOString())
   ```

3. **Compute dashboard stats:**
   - `totalSalesCents`: sum of all todayOrders total_cents
   - `orderCount`: todayOrders.length
   - `posCount`: orders where channel === 'pos'
   - `onlineCount`: orders where channel === 'online'

4. **Low stock query:**
   Fetch all active products and filter client-side (per Research recommendation for small catalogs):
   ```typescript
   const { data: products } = await supabase
     .from('products')
     .select('name, stock_quantity, reorder_threshold')
     .eq('store_id', storeId)
     .eq('is_active', true)

   const lowStockProducts = (products ?? []).filter(
     p => p.stock_quantity <= p.reorder_threshold
   )
   ```

5. **Layout per D-01:**
   - Page title: "Dashboard" (`<h1>` with `font-sans font-bold text-xl`)
   - Hero cards in a grid: `grid grid-cols-1 md:grid-cols-3 gap-lg` (24px gap)
   - Card 1: label="Today's Sales", value=formatNZD(totalSalesCents)
   - Card 2: label="Orders Today", value=orderCount.toString()
   - Card 3: label="POS vs Online", value=`${posCount + onlineCount}`, subLabel=`POS: ${posCount} / Online: ${onlineCount}`
   - Below cards: `<LowStockAlertList products={lowStockProducts} />`

6. **No trend charts, no recent orders feed** per D-02 and D-04.
