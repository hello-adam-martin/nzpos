**2. Create src/actions/promos/validatePromoCode.ts:**

```typescript
'use server'
import 'server-only'
import { headers } from 'next/headers'
import { checkRateLimit } from '@/lib/rateLimit'
```

- This is a PUBLIC action (no auth required — storefront customers call it).
- Use admin client (createSupabaseAdminClient) since storefront has no auth session.
- Rate limiting per DISC-02 and D-18:
  ```typescript
  const headersList = await headers()
  const ip = headersList.get('x-forwarded-for')?.split(',')[0]?.trim() ?? 'unknown'
  if (!checkRateLimit(ip, 10)) {
    return { error: 'rate_limited', message: 'Too many attempts. Try again in a moment.' }
  }
  ```
- Input: `{ code: string; cartTotalCents: number; storeId: string }`
- Query promo_codes: `.select('*').eq('store_id', storeId).eq('code', code.toUpperCase()).eq('is_active', true).single()`
- Validation checks with specific error messages per D-17:
  - No result: `{ error: 'invalid', message: 'That code is invalid or has expired. Double-check the code or contact us for help.' }`
  - Expired (`expires_at && new Date(expires_at) < new Date()`): `{ error: 'expired', message: 'That code is invalid or has expired. Double-check the code or contact us for help.' }`
  - Max uses reached (`max_uses && current_uses >= max_uses`): `{ error: 'max_uses', message: 'That code is invalid or has expired. Double-check the code or contact us for help.' }`
  - Minimum order not met (`min_order_cents > 0 && cartTotalCents < min_order_cents`): `{ error: 'min_order', message: 'Minimum order NZ$X.XX required.' }` — use formatCents for the amount.
- On success, calculate discount:
  - If percentage: `discountCents = Math.floor(cartTotalCents * discount_value / 100)`
  - If fixed: `discountCents = discount_value`
  - Clamp: `discountCents = Math.min(discountCents, cartTotalCents)` (discount cannot exceed cart total)
- Return: `{ success: true, promoId: promo.id, discountCents, discountType: promo.discount_type, code: promo.code }`
