## Spacing Scale

Declared values (all multiples of 4). Source: `globals.css` `:root` block.

| Token | Value | CSS Var | Usage |
|-------|-------|---------|-------|
| xs | 4px | `var(--space-xs)` | Icon gaps, inline padding, bullet gaps |
| sm | 8px | `var(--space-sm)` | Compact element spacing, list item gaps |
| md | 16px | `var(--space-md)` | Default element spacing, section px padding |
| lg | 24px | `var(--space-lg)` | Section padding, card grid gaps |
| xl | 32px | `var(--space-xl)` | Card internal padding, layout gaps |
| 2xl | 48px | `var(--space-2xl)` | Major section vertical padding (tight sections) |
| 3xl | 64px | `var(--space-3xl)` | Page section vertical padding (all full sections) |

Exceptions: none for this phase. No POS touch targets needed — marketing pages only.

## Typography

Source: DESIGN.md typography scale. Only the sizes actually used in marketing page components are declared here.

| Role | Size | CSS Class | Weight | Line Height | Font |
|------|------|-----------|--------|-------------|------|
| Body | 16px (1rem) | `text-base` | 400 (`font-sans`) | 1.5 | DM Sans |
| Label / caption | 14px (0.875rem) | `text-sm` | 400–600 | 1.5 | DM Sans |
| Section heading | 24px (1.5rem) | `text-2xl` | 700 (`font-bold`) | 1.3 | DM Sans |
| Hero / display | 48px (3rem) | `text-5xl` | 700 (`font-bold`) | 1.1 | Satoshi (`font-display`) |

Notes:
- Hero headings: `font-display text-3xl md:text-5xl font-bold leading-[1.1]` — matches xero/page.tsx exactly
- Section headings use `font-sans` (not `font-display`) at `text-2xl font-bold leading-[1.3]` — matches established pattern
- Uppercase labels (section eyebrows): `text-sm font-bold uppercase tracking-wide` in `var(--color-text-muted)` or accent color
- Price display (add-on cards): `text-sm font-bold text-[var(--color-amber)]`
- Hero subtext: `text-base md:text-lg text-white/70` under navy hero backgrounds

## Color

Source: DESIGN.md, globals.css `@theme` block.

| Role | Value | CSS Var | Usage |
|------|-------|---------|-------|
| Dominant (60%) | #FAFAF9 (warm stone white) | `var(--color-bg)` | Main page background for alternate sections |
| Secondary (30%) | #F5F5F4 (light stone) | `var(--color-surface)` | Features section background (alternating with bg) |
| Card | #FFFFFF | `var(--color-card)` | Add-on cards, before/after "With" panel, free plan card |
| Navy (hero/CTA) | #1E293B | `var(--color-navy)` | Hero sections, CTA sections, step number circles |
| Accent (10%) | #E67E22 (amber) | `var(--color-amber)` | Reserved for specific elements listed below |
| Destructive | #DC2626 | `var(--color-error)` | Destructive actions only — not used in this phase |
| Success | #059669 | `var(--color-success)` | Checkmark icons in "With" column, free plan bullets |

Accent (`var(--color-amber)`) reserved for:
- "Learn more →" link text on add-on cards
- Price text on add-on cards (`$X/month NZD`)
- Primary CTA buttons (`Get started free`)
- "With [Add-on]" panel border and eyebrow label
- Compliance callout box — amber accent border or background treatment
- Hover border on add-on cards (`hover:border-[var(--color-amber)]`)

Do NOT use accent for: body text, section headings, background fills outside the defined elements above.

## Component Inventory

Components to create or modify in this phase. Executor must follow these specs exactly.

### New: Add-On Detail Pages (3 pages)

All 3 pages follow the xero/page.tsx section structure verbatim:

**Section 1 — Hero** (`bg-[var(--color-navy)]`)
- Back link: `← Back to add-ons` in `text-sm text-white/50 hover:text-white` — left-aligned
- H1: `font-display text-3xl md:text-5xl font-bold text-white leading-[1.1]`
- Subtext: `font-sans text-base md:text-lg text-white/70 mt-[var(--space-md)] max-w-2xl`
- CTA row: amber `Get started free` button + `text-sm text-white/50` price note

**Section 2 — Before/After** (`bg-[var(--color-bg)]`)
- Two-column grid (`grid-cols-1 md:grid-cols-2 gap-[var(--space-lg)]`)
- "Without" panel: `border border-[var(--color-border)] rounded-lg p-[var(--space-xl)]` — X icons in `var(--color-text-light)`
- "With" panel: `border-2 border-[var(--color-amber)] rounded-lg p-[var(--space-xl)] bg-white` — checkmark icons in `var(--color-success)`

**Section 3 — Compliance Callout** (NEW — between Before/After and Features, only on relevant pages)
- Gift Cards: NZ Fair Trading Act 2024 — 3-year minimum expiry rule
- Loyalty Points: Privacy Amendment Act 2025, IPP 3A consent
- Advanced Reporting: COGS/margin tracking context (not a legal act, contextual callout)
- Visual: `border border-[var(--color-amber)] rounded-lg p-[var(--space-xl)] bg-[var(--color-navy)]` with amber icon (shield SVG at 24px) + white heading + `text-white/70` body
- Sits between Before/After and Features sections — distinct background break

**Section 4 — Features** (`bg-[var(--color-surface)]`)
- H2: `font-sans text-2xl font-bold text-[var(--color-text)] leading-[1.3]` — "What you get"
- Grid: `grid-cols-1 sm:grid-cols-2 gap-[var(--space-lg)] mt-[var(--space-xl)]`
- Feature item: bold title at `text-base font-bold` + `text-sm text-[var(--color-text-muted)]` description
- Feature count: 4–6 items per page (Claude's discretion per CONTEXT.md)

**Section 5 — How It Works** (`bg-[var(--color-bg)]`)
- H2: "Three steps. Two minutes." — `font-sans text-2xl font-bold`
- Grid: `grid-cols-1 md:grid-cols-3 gap-[var(--space-lg)]`
- Step: circle `w-10 h-10 rounded-full bg-[var(--color-navy)]` with white bold number, title, description

**Section 6 — CTA** (`bg-[var(--color-navy)]`)
- Centered layout, amber `Get started free` button
- Price note in `text-white/70` below button
