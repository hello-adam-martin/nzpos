**A. src/actions/gift-cards/listGiftCards.ts:**

```typescript
'use server'
import 'server-only'
```

1. Zod input schema:
   ```typescript
   {
     page: z.number().int().positive().default(1),
     pageSize: z.number().int().positive().max(100).default(20),
     status: z.enum(['all', 'active', 'redeemed', 'expired', 'voided']).default('all'),
     search: z.string().optional(),  // last 4 digits search
     sortBy: z.enum(['issued_at', 'expires_at', 'balance_cents', 'original_value_cents']).default('issued_at'),
     sortOrder: z.enum(['asc', 'desc']).default('desc'),
   }
   ```

2. Auth via `resolveStaffAuth()` — get store_id
3. Query `gift_cards` table with filters:
   - `.eq('store_id', storeId)`
   - If status !== 'all': `.eq('status', status)` — but also compute effective expired status for 'active' cards past expiry (Pitfall 2)
   - If search: `.like('code', `%${search}`)` (search by last digits)
   - Order by sortBy + sortOrder
   - Range for pagination: `.range((page-1)*pageSize, page*pageSize - 1)`
4. Also query count for pagination
5. Return `{ data: GiftCardRow[], total: number, page, pageSize }`
6. Each row includes: id, code (last 4 only for display: `****-${code.slice(-4)}`), original_value_cents, balance_cents, status (computed effective), issued_at, expires_at

**B. src/actions/gift-cards/getGiftCard.ts:**

```typescript
'use server'
import 'server-only'
```

1. Zod input: `{ giftCardId: z.string().uuid() }`
2. Auth via `resolveStaffAuth()`
3. Fetch gift card by id + store_id
4. Fetch related redemptions from `gift_card_redemptions` ordered by `redeemed_at ASC`
5. For each redemption, join staff name if staff_id present
6. Return full gift card data + code (full 8-digit formatted) + redemptions array
7. Compute effective status (active + past expiry = expired)

**C. src/actions/gift-cards/voidGiftCard.ts:**

```typescript
'use server'
import 'server-only'
```

1. Zod input: `{ giftCardId: z.string().uuid(), reason: z.string().min(4, 'Reason must be at least 4 characters') }`
2. Auth via `resolveStaffAuth()` — also verify `role === 'owner'` (D-14: owner-only action)
3. Call `supabase.rpc('void_gift_card', { p_store_id: storeId, p_gift_card_id: giftCardId, p_reason: reason })`
4. If error, return `{ success: false, error: 'Failed to void gift card' }`
5. Return `{ success: true }`
6. Revalidate `/admin/gift-cards` path
