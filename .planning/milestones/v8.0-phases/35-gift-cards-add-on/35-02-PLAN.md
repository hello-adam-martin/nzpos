**A. src/actions/gift-cards/issueGiftCard.ts — POS issuance (D-04 POS path):**

```typescript
'use server'
import 'server-only'
```

1. Zod schema: `{ valueCents: z.number().int().positive(), buyerEmail: z.string().email().optional() }`
2. Call `resolveStaffAuth()` for auth — get staff.store_id and staff.staff_id
3. Check feature gate: query `store_plans` for `has_gift_cards = true` on the store. If not enabled, return error.
4. Generate unique 8-digit numeric code (D-01):
   ```typescript
   async function generateUniqueCode(storeId: string, supabase: SupabaseClient): Promise<string> {
     for (let attempt = 0; attempt < 10; attempt++) {
       const code = String(Math.floor(Math.random() * 100_000_000)).padStart(8, '0')
       const { data } = await supabase
         .from('gift_cards')
         .select('id')
         .eq('store_id', storeId)
         .eq('code', code)
         .maybeSingle()
       if (!data) return code
     }
     throw new Error('Failed to generate unique gift card code after 10 attempts')
   }
   ```
5. Call `supabase.rpc('issue_gift_card', { p_store_id, p_code, p_value_cents: valueCents, p_channel: 'pos', p_buyer_email: buyerEmail })` via admin client
6. If buyerEmail provided, fire-and-forget email: `void sendEmail({ to: buyerEmail, subject: 'Your gift card from ${storeName}', react: <GiftCardEmail ... /> })`
7. Fetch store name from stores table for email and return value
8. Return `{ success: true, code, valueCents, expiresAt }` — formatted code with dash for display: `${code.slice(0,4)}-${code.slice(4)}`
