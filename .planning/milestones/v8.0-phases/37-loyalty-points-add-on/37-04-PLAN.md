**3. Earn loyalty points on net amount:**

Look up customer by email (from `session.customer_details?.email`). Pass `session.amount_total` as `p_net_amount_cents` -- Stripe `amount_total` already reflects all negative line items (gift card deduction + loyalty deduction), so it IS the net amount. Do NOT compute a separate `netAmountCents` by subtracting gift card or loyalty amounts again.

```typescript
// Loyalty points earning — look up customer by email
const customerEmail = session.customer_details?.email
if (customerEmail) {
  const { data: customer } = await supabase
    .from('customers')
    .select('id')
    .eq('store_id', storeId)
    .eq('email', customerEmail)
    .maybeSingle()

  if (customer) {
    // D-09: Stripe amount_total already reflects all negative line items
    // (gift card + loyalty deductions). It IS the net amount paid.
    // Do NOT subtract gift card or loyalty amounts again.
    const netAmountCents = session.amount_total ?? 0

    if (netAmountCents > 0) {
      try {
        await supabase.rpc('earn_loyalty_points', {
          p_store_id: storeId,
          p_customer_id: customer.id,
          p_order_id: orderId,
          p_net_amount_cents: netAmountCents,
          p_channel: 'online',
          p_staff_id: null,
        })
      } catch (err) {
        console.warn('[stripe-webhook] Loyalty earning failed (non-fatal):', err)
      }
    }
  }
}
```
