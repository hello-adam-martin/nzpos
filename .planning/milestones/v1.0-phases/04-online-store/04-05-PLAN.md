Per research Pattern 3 and D-08:
```typescript
'use server'
import 'server-only'
import { stripe } from '@/lib/stripe'
import { createSupabaseAdminClient } from '@/lib/supabase/admin'
import { calcLineItem, calcOrderGST } from '@/lib/gst'
```

Input: `{ items: { productId: string; quantity: number }[]; promoId?: string; promoDiscountCents?: number; promoDiscountType?: 'percentage' | 'fixed' }`

NOTE: The field is `promoId` (matching validatePromoCode return type), NOT `promoCodeId`.

Logic:
1. Get storeId from STORE_ID env var.
2. Re-fetch product prices from DB — NEVER trust client prices (Pitfall 6). Query products by IDs from `items`, verify `is_active=true`, verify `stock_quantity >= requested quantity`.
3. If any product out of stock: return `{ error: 'out_of_stock', productName: '...' }`.
4. Build cart with GST: For each item, calcLineItem(price_cents, quantity, 0) to get lineTotal and gst.
5. If promoId provided: re-validate promo from DB (double-check expiry, uses, active). Calculate discount. Distribute pro-rata across lines (same algorithm as applyCartDiscount: Math.floor share for all, last item absorbs remainder). Recalculate GST per-line on discounted amounts.
6. Calculate totals: totalCents = sum of line totals after discount. gstCents = sum of line GSTs.
7. Create PENDING order record in Supabase:
   ```typescript
   const { data: order } = await supabase.from('orders').insert({
     store_id: storeId, channel: 'online', status: 'pending',
     subtotal_cents: subtotalBeforeDiscount, gst_cents: gstCents,
     total_cents: totalCents, discount_cents: totalDiscountCents,
     payment_method: 'stripe',
   }).select('id').single()
   ```
8. **CRITICAL: Insert order_items for each cart item.** This is required for complete_online_sale RPC to have items to decrement stock for:
   ```typescript
   const orderItemsData = cartItems.map(item => ({
     order_id: order!.id,
     store_id: storeId,
     product_id: item.productId,
     product_name: item.productName,
     unit_price_cents: item.unitPriceCents,
     quantity: item.quantity,
     discount_cents: item.discountShareCents,
     line_total_cents: item.lineTotalCents,
     gst_cents: item.gstCents,
   }))
   const { error: itemsError } = await supabase.from('order_items').insert(orderItemsData)
   if (itemsError) throw itemsError
   ```
9. If promo applied: increment current_uses on promo_codes table.
10. Create Stripe Checkout Session:
    ```typescript
    const session = await stripe.checkout.sessions.create({
      mode: 'payment', currency: 'nzd',
      line_items: cartItems.map(item => ({
        price_data: {
          currency: 'nzd',
          product_data: { name: item.productName },
          unit_amount: item.lineTotalCents,  // GST-inclusive, already includes discount share
        },
        quantity: 1,  // quantity factored into lineTotalCents
      })),
      metadata: { order_id: order!.id, store_id: storeId },
      success_url: `${process.env.NEXT_PUBLIC_BASE_URL}/order/${order!.id}/confirmation?session_id={CHECKOUT_SESSION_ID}`,
      cancel_url: `${process.env.NEXT_PUBLIC_BASE_URL}/`,
    })
    ```
    Note: Do NOT set tax_rates or Stripe Tax — prices are GST-inclusive (Pitfall per research).
11. Return `{ url: session.url }`.
