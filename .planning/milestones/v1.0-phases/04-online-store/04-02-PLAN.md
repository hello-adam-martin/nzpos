**Types (separate from POS CartItem — do NOT import from cart.ts per D-06 pattern):**
```typescript
export type StoreCartItem = {
  productId: string
  productName: string
  unitPriceCents: number
  quantity: number
  imageUrl: string | null
  slug: string | null
  maxStock: number  // tracks available stock for sold-out check
}
```

Note: lineTotalCents and gstCents are NOT stored on the item — they are computed dynamically from unitPriceCents * quantity using calcLineItem. This avoids stale GST if the formula changes.

```typescript
export type StoreCartState = {
  items: StoreCartItem[]
  promoCode: string | null
  promoDiscountCents: number
  promoDiscountType: 'percentage' | 'fixed' | null
  isDrawerOpen: boolean
}
```

**Actions:**
```typescript
export type StoreCartAction =
  | { type: 'ADD_ITEM'; product: { id: string; name: string; priceCents: number; imageUrl: string | null; slug: string | null; stockQuantity: number } }
  | { type: 'SET_QUANTITY'; productId: string; quantity: number }
  | { type: 'REMOVE_ITEM'; productId: string }
  | { type: 'APPLY_PROMO'; code: string; discountCents: number; discountType: 'percentage' | 'fixed' }
  | { type: 'CLEAR_PROMO' }
  | { type: 'CLEAR_CART' }
  | { type: 'TOGGLE_DRAWER' }
  | { type: 'OPEN_DRAWER' }
  | { type: 'CLOSE_DRAWER' }
  | { type: 'HYDRATE'; payload: StoreCartState }
```

**Reducer logic:**
- ADD_ITEM: If product already in cart, increment quantity (up to maxStock). If not, add with quantity 1. Per STORE-08: reject add if stockQuantity <= 0.
- SET_QUANTITY: Clamp to [1, maxStock]. If 0, remove item.
- REMOVE_ITEM: Filter out by productId.
- APPLY_PROMO: Set promoCode, promoDiscountCents, promoDiscountType.
- CLEAR_PROMO: Reset promo fields to null/0.
- CLEAR_CART: Reset to initial state (keep drawer closed).
- TOGGLE/OPEN/CLOSE_DRAWER: Control drawer visibility.
- HYDRATE: Replace state from localStorage (except isDrawerOpen — always starts closed).

**localStorage persistence per D-07:**
- CART_STORAGE_KEY = 'nzpos_store_cart'
- useEffect on mount: read from localStorage, dispatch HYDRATE if data exists. Wrap in try/catch (corrupt storage = ignore).
- useEffect on state change: write to localStorage. Exclude isDrawerOpen from persisted data.

**useCart hook:**
Export a `useCart()` hook that returns:
```typescript
{
  state: StoreCartState
  dispatch: React.Dispatch<StoreCartAction>
  itemCount: number           // total quantity across all items
  subtotalCents: number       // sum of unitPriceCents * quantity for all items
  discountCents: number       // promoDiscountCents
  totalCents: number          // subtotalCents - discountCents (min 0)
  gstCents: number            // calcOrderGST on per-line GST values after discount
}
```

Computed values (itemCount, subtotalCents, gstCents, totalCents) should use `useMemo` for performance.

For GST calculation with promo discount per D-19: distribute discount pro-rata across lines (same algorithm as applyCartDiscount in cart.ts), then calcLineItem per line on discounted amount, then calcOrderGST. Implement this inline — do NOT import applyCartDiscount from cart.ts (different CartItem type).

**CartProvider component:**
Wraps children in CartContext.Provider with value = { state, dispatch, ...computed }.
